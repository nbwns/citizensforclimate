Bienvenue sur CitizensForClimate.be !

Notre mouvement citoyen spontané, né dans la foulée des récentes mobilisations massives pour le climat dans le monde, a pour objectif de vous aider, vous citoyen du monde présent sur le territoire belge, à vous mobiliser en masse pour le climat !

Comment ? En centralisant les actions dans lesquels vous pouvez vous investir : rassemblements, pétitions, boycotts, actions en justice, lettres ouvertes, sittings, éducation permanente, zones à défendre, actions directes non violentes, etc. Toute action à DIMENSION COLLECTIVE est bienvenue sur notre site de mobilisation civile.

Tu organises ou fait partie d’une institution qui organise une action pour le climat, encode là dans ce formulaire pour qu’elles soit référencées sur le site de centraliser les différentes actions organisées dans notre pays

Que fais-tu le 2 décembre ? Un repas de famille, une virée dans les Ardennnes ? Tu peux pas, t’as climat ! Viens à la grande marche organisée à Bruxelles : http://claimtheclimate.be/
